# GenColorPickerTableCells.py

Generate color picker table cells for Qt5's table widget.

![Screenshot](screenshot.gif)

## Running
> Note: Python version >=3.7.0 is required.

With Python 3

```shell
python GenColorPickerTableCells.py
```
